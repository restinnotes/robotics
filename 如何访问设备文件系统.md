## 当前问题

设备在Linux上只显示为串口设备（`/dev/ttyACM0`），无法像Windows那样直接访问文件系统。

## 可能的解决方案

### 方案1: 检查设备是否有模式切换

运行以下命令检查设备当前模式：

```bash
lsusb -v -d 108c:ab3c | grep -i "class\|interface"
```

### 方案2: 尝试USB模式切换

某些设备需要切换到Mass Storage模式：

```bash
# 查看当前设备信息
lsusb | grep APP

# 尝试模式切换（需要root）
sudo usb_modeswitch -v 0x108c -p 0xab3c
```

切换后检查是否出现新的存储设备：

```bash
lsblk
```

### 方案3: 检查是否有Bosch工具

查找Bosch Sensortec相关的工具：

```bash
# 搜索已安装的Bosch相关工具
dpkg -l | grep -i bosch
which bhi3xx  # 或其他可能的工具名
```

### 方案4: 通过串口发送特定命令

如果设备支持文件系统命令，可能需要特定的协议。请查看设备文档。
